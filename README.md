Frontend Basics
HTML Basics

HTML (HyperText Markup Language) is a markup language, not a programming language.
It is used to structure and organize content on web pages.

HTML defines the layout of elements such as text, images, links, headings, and other webpage components.

Creating an HTML Document

Most code editors provide shortcuts that automatically generate the basic HTML document structure. This helps developers start building webpages quickly.

Structure of an HTML Document

An HTML document mainly contains two primary sections:

Head Section

The head section contains metadata about the webpage.
Important metadata elements include:

Character encoding (charset) – defines how characters are stored and displayed.

Viewport settings – helps make the webpage responsive on different screen sizes such as mobile devices and tablets.

Body Section

The body section contains the visible content of the webpage.

Examples of content placed in the body include:

Text

Headings

Images

Links

Buttons

Other webpage elements

Everything that appears on the webpage is placed inside the body.

HTML and Presentation

HTML mainly focuses on structure rather than visual presentation.

Web browsers apply their own default styling to HTML elements through something called a User Agent Stylesheet.

Because each browser has its own stylesheet, the appearance of elements may vary slightly across different browsers.

Example behavior:

Unvisited links typically appear blue

Visited links may appear purple or another color depending on the browser

HTML Tags

Most HTML elements consist of:

An opening tag

Content inside the element

A closing tag

If an element contains data or content, it usually requires a closing tag.

Block-Level Elements

Block-level elements:

Start on a new line

Occupy the full available width

This is why heading elements automatically appear on separate lines.

Inline Elements

Inline elements:

Occupy only the space required

Do not start on a new line

Images and links are common examples of inline elements.

CSS Basics

CSS (Cascading Style Sheets) is used to style and design HTML elements.

CSS works using rules, and each rule targets HTML elements using a selector.

CSS Box Model

In CSS, every HTML element is treated as a box.

Understanding the Box Model is important because it controls spacing and layout of elements.

The box model consists of four main parts:

Content – the actual element content.

Padding – the space between content and border.

Border – the boundary around the element.

Margin – the space outside the border.

Padding

Padding is the space between the content and the border.

Padding exists on four sides:

Top

Right

Bottom

Left

Padding increases space inside the element.

Margin

Margin is the space outside the border.

Like padding, margin also exists on four sides.

Margin is mainly used to create space between elements.

Alignment

Elements can be aligned in different ways, including:

Left alignment

Right alignment

Center alignment

Margins can also be used to help center elements.
